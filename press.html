Pitch produit

Le probleme

Un joueur CS2 ambitieux a 2 options pour progresser : payer 30-80 EUR/h un coach humain (cher, non scalable) ou se perdre dans des stats brutes (Leetify, csstats) sans contexte tactique. Personne ne lui dit pourquoi il a perdu un round specifique en termes actionables.

La solution FragValue

Un Coach IA contextuel qui analyse chaque demo round par round, cite des moments precis ("[R14] tu peek long sans util alors que l'adversaire a 2 flashs"), et produit un diagnostic structure sur 11 axes (aim, crosshair, spray, utility, positioning, gamesense, economy, mental, movement, comms, reaction).

Le moat technique

Premier produit avec auto-sync FACEIT Downloads API officielle (acces partenariat mai 2026). Match termine -> 5-10 min plus tard l'analyse est dans la boite. Zero friction vs concurrents qui demandent un upload manuel.

Le modele economique

Freemium : Free (5 analyses/mois + 4 diagnostics IA), Pro 9 EUR/mois (7 jours essai gratuit, illimite + 20 Coach IA msg/jour + auto-sync), Elite 25 EUR/mois (50 msg/jour + outils equipe pour 5 joueurs), Lifetime Pro 99 EUR one-time (50 places launch). Sans engagement, annulation 1 clic, 14j satisfait ou rembourse.

Le concurrence

Leetify et csstats : excellent sur les stats brutes, faible sur le coaching contextuel + zero IA. Coachs humains : tres bons mais non scalables (15-80 EUR/h, dispos limitees). FragValue cible le creneau "Coach IA scalable + contextuel" entre les deux.

FAQ presse

En quoi FragValue se differencie de Leetify, csstats, scope.gg ?
3 differences cles : (1) Coach IA contextuel par round qui cite les ticks et explique les decisions tactiques, pas juste des stats. Les concurrents montrent ADR/KAST sans dire pourquoi tu as perdu le round 14. (2) Auto-sync FACEIT officiel via la Downloads API (acces partenariat mai 2026) : match termine -> analyse pushed en notif sans aucune action manuelle. (3) 11 axes de progression structures au lieu d'un score unique : tu sais exactement quel axe travailler en priorite.
Quelle IA derriere le Coach ?
Claude d'Anthropic (modele Opus 4.7, version 1M tokens context). Le Coach IA recoit en contexte : le round-by-round complet (kills, bombs, grenades, callouts), les stats du joueur, les benchmarks pros HLTV, et son historique des diagnostics precedents (axe 9 "Suivi progression"). Cela permet un coaching cumulatif vs analyses one-shot.
Quel scope FACEIT API utilisez-vous ?
FragValue utilise la Data API (lookup joueurs, match history) en gratuit + la Downloads API (signed URL pour fichiers .dem) en partenariat valide. Le partenariat Downloads API est gated par une application form 30j (fce.gg/downloads-api-application), validee pour FragValue en mai 2026 par l'equipe FACEIT partnerships.
Combien d'utilisateurs ?
Donnees pilote en cours, beta privee jusqu'au public launch (estime juin 2026). Contact press@fragvalue.com pour les chiffres exacts a la date de ton article + une demo Pro/Elite gratuite 1 mois pour tester.
FragValue est-il legal vis-a-vis de FACEIT / Valve ?
100% conforme. Nous utilisons les APIs officielles FACEIT (Data + Downloads) sous accord partenariat, et le format demo CS2 public (.dem) parse cote serveur avec demoparser2 (open source). Aucune injection in-game, aucun scraping non-autorise. Conditions Generales : fragvalue.com/cgv.
