LГ©gende :

- `вҳ…` = CTA principal
- `в–ҫ` = dropdown
- `в—ү` = bouton actif / sГ©lectionnГ©

## Layout patterns

**Grille de cards** (utilisГ©e pour pro-demos, lineup-library) :
- `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`
- Card hover : `transform: translateY(-2px)` + `border-color: accent`

**Modal dГ©tail** (lineup, pro match) :
- Backdrop `rgba(0,0,0,.7)` + `backdrop-filter: blur(8px)`
- Content `max-width: 640px`, centrГ©
- Close via click backdrop, ESC ou X

**Paywall soft** :
- Section visible mais overlay gradient qui masque le bas
- Texte "DГ©bloque avec Pro" + CTA upgrade
- Utile pour teaser la valeur sans cacher 100% du contenu
