fragvalue-mcp
v0.1.0 · Beta · MIT License

4 tools read-only exposes aux agents IA. Resolution par FACEIT nickname ou Supabase user_id, validation Zod, communication STDIO/JSON-RPC standard MCP.

list_recent_demos
Historique des N dernieres demos d'un joueur avec FV Rating, map, rounds, kills, date.
get_demo_analysis
Analyse complete d'une demo par UUID. FV Rating, scoreboard, statut parsing, lien partage.
get_player_stats
Stats agregees sur N dernieres demos. FV Rating moyen / min / max, tendance, maps preferees.
search_articles
Recherche dans le blog FragValue par mot-cle. Titres, tags, keywords ponderes.

// CAS D'USAGE

Pour qui c'est fait.

3 profils de developpeurs et builders utilisent deja le MCP server FragValue en beta.

Coach CS2
Rapport hebdo automatique
Tu coach 10-30 students CS2. Tu connectes Claude Desktop au MCP server FragValue. Tu dis "fais-moi le rapport semaine 24 de mes students" et Claude appelle list_recent_demos pour chaque, calcule les tendances, te formate un Discord post pret a copier-coller. Gain : 30 min par semaine.
Streamer Twitch
Stats live dans le chat
Tu build un bot Twitch en Cursor qui ecoute les commandes !stats. Le bot appelle get_player_stats sur ton compte FragValue et balance les stats dans le chat en 200ms. Plus besoin de hardcoder une commande, l'agent fait la requete et formate.
Org esports T3
Scouting de joueurs
Tu cherches un nouveau joueur pour ton academy team. Tu dis a Claude "compare le FV Rating moyen de Player_A et Player_B sur leurs 50 dernieres demos, regarde leur tendance et leur map preferee". Decision en 30 secondes au lieu d'1h de spreadsheet HLTV.

// AUTRES SURFACES

Plus que MCP.

Le MCP server est la premiere etape. Voici ce qui arrive ensuite pour offrir une surface API complete.

REST API · Q3 2026
Endpoints HTTP classiques avec OAuth ou API key. Pour les devs qui construisent des apps custom (Discord bots, Twitch overlays, mobile apps). Plans Free 100/mo, Pro 10k/mo, Enterprise unlimited.
Webhooks · Q4 2026
Recoit des events FragValue dans ton service. Quand un user analyse une demo, ta plateforme recoit demo.completed avec les stats. Use case : Discord bots, dashboards externes, integration CRM esports.
SDK npm · 2027
@fragvalue/sdk en TypeScript. 1 ligne pour integrer FragValue dans n'importe quelle app Node.js. Auto-completion, types, abstraction au-dessus de l'API REST + MCP.
Embeddable widgets · 2027
Iframe avec ton FV Rating live + KPIs, integrable dans n'importe quel site (HLTV-like). Pour les sites de coaching, les profils joueurs, les pages de streamers.
