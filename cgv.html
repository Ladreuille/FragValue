Article 3 - Description des services

FragValue est une plateforme web SaaS d'analyse de donnees pour le jeu video Counter-Strike 2 (CS2). Les services proposes incluent :

  • Plan Free (gratuit) : analyse de matchs FACEIT limitee (3 par mois), statistiques basiques, 1 diagnostic IA par mois.
  • Plan Pro (paye) : analyses illimitees, KPIs avances, 2D Replay, Match Report round-by-round, Coach IA illimite, Pro demos viewer, comparaison multi-joueurs, export PDF, support prioritaire.
  • Plan Team (paye) : tout Pro + dashboard equipe, analyse tactique par round, preparation veto, anti-strat tool, pro benchmarks, pro strats DB, coaching mensuel, acces API.

Les fonctionnalites detaillees de chaque plan sont presentees sur la page tarifs. Le Vendeur se reserve le droit de modifier la composition des plans, en preservant la valeur globale pour le Client.
