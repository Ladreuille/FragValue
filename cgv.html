Article 3 - Description des services

FragValue est une plateforme web SaaS d'analyse de données pour le jeu video Counter-Strike 2 (CS2). Les services proposes incluent :

  • Plan Free (gratuit) : analyse de matchs FACEIT limitée (3 par mois), statistiques basiques, 1 diagnostic Coach IA par mois.
  • Plan Pro (paye) : analysés illimitees, KPIs avancés, 2D Replay, Match Report round-by-round, Coach IA illimite, Pro demos viewer, comparaison multi-joueurs, export PDF, support prioritaire.
  • Plan Elite (paye, destine aux coachs et equipes) : tout Pro + dashboard equipe, analyse tactique par round, préparation de véto, anti-strat tool, pro benchmarks, pro strats DB et fonctionnalites exclusives pour le coaching.

Les fonctionnalites détaillées et tarifs de chaque plan sont présentes sur la page tarifs. Le Vendeur se réserve le droit de modifier la composition des plans, en préservant la valeur globale pour le Client.
