# Twitter content calendar - 7 jours post-launch

7 tweets pré-rédigés, format aligné avec ton style (no emoji, accents OK,
sans em dash). Programme un par jour pendant 7 jours pour maintenir
l'engagement post-launch sans avoir à réfléchir à chaque tweet.

**Outil de scheduling** : Twitter natif (icône calendrier dans la box de tweet),
ou Buffer / Hootsuite / TweetDeck pour batcher.

## Conseils tactiques

- **Heure optimale FR** : 12h-14h (pause dej) ou 19h-22h (apres ecole/boulot)
- **Espace de 24h minimum** entre 2 tweets pour ne pas saturer ta TL
- **Engage avec les replies** : answer les questions dans les 4h suivant le tweet (boost l'algo)
- **Pin** le tweet du Jour 5 (Discord push) pendant 48h pour drive le traffic vers la commu
- **Replace** les `N` du Jour 7 par les vraies stats du dashboard interne `/admin/dashboard.html`

---

## Apres ces 7 jours

Continue le rythme avec :
- 1 insight stat / semaine (sourced depuis ton dashboard ou tes propres analyses)
- 1 behind-the-scenes / semaine (technique / philosophie produit)
- 1 user testimonial / semaine (quand tu en collectes)

3 tweets/semaine est largement suffisant pour growth organique sans
spammer. Quality > quantity.
